# Vartalap   
A secure, real-time chat application built using the MERN stack. Vartalap enables users to communicate instantly with end-to-end encrypted messaging, smooth UI, and reliable WebSocket connectivity.

##Features
- End-to-end encrypted messaging  
- Real-time chat using Socket.io  
- One-to-one and group conversations  
- Modern and responsive UI  
- Real-time online/offline status  

##Tech Stack
**Frontend:** React, TailwindCSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Real-time:** Socket.io  
**Auth:** JWT
